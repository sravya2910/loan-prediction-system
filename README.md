# Loan Prediction System using CIBIL Score

## 📌 Project Overview
This project predicts whether a loan will be approved based on applicant details and CIBIL score using Machine Learning.

## 🚀 Features
- Predict loan approval
- Uses Random Forest Algorithm
- Includes Streamlit web app

## 📊 Dataset Features
- Income
- Loan Amount
- CIBIL Score
- Credit History

## 🧠 Model Used
- Random Forest Classifier

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run ML model
cd src
python loan_prediction.py

### 3. Run Web App
cd app
streamlit run app.py

## 📈 Output
- Loan Approved / Rejected prediction
